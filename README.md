# Flow

Aplicație wellness: nutriție, exerciții, rețete, social. Monorepo: **website** (Next.js), **app** (Flutter), **docs**.

## Structură

- **website/** – landing + dashboard web (Next.js, Supabase)
- **docs/** – setup, referințe, backlog (vezi [docs/README.md](docs/README.md))

## Start rapid

- **Web:** `cd website && npm install && npm run dev` → http://localhost:3000
- **Setup Supabase, Realtime, Recipes, Exercises:** [docs/SETUP.md](docs/SETUP.md)
